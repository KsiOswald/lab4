# Задание 1


## Задача 1


## Текст задачи


Дерево содержит строки. Добавить к каждой строке заданный символ в конец. 


### Алгоритм решения


Количество строк в дереве, их длина и каждый символ создается случайным образом.
Каждая строка сначала вставляется в список. Затем из списка строк строится дерево.
Бинарное дерево строится путем вставки нового элемента в пустой узел
Если значение текущего узла меньше чем новый, то происходит переход на левую ветвь
В ином случае (больше или равно), происходит переход на правую ветвь
Полученное дерево выводится на экран
Затем, на основе полученного дерева строится новое, путем добавления к каждой строке в вершине полученного от пользователя символа
После этого новое дерево выводится на экран

Если значение текущего узла меньше чем новый, то происходит переход на левую ветвь
В ином случае (больше или равно), происходит переход на правую ветвь
Полученное дерево выводится на экран
